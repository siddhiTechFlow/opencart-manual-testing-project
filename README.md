# NopCommerce Testing Project

This project contains manual testing of the NopCommerce demo e-commerce website.

## Objective
To validate the functionality of the application by creating and executing test cases.

## Testing Activities
- Test case design
- Test execution
- Validation testing
- Negative testing
- Boundary testing

## Modules Tested
- Login
- Registration
- Product Search
- Product Display

## Tools Used
- MS Excel
- Browser
- ChatGPT (for test scenarios)

## Files Included
- Test cases
- Test scenarios
- Test results
